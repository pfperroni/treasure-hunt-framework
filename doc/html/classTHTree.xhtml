Detailed Description

Treasure Hunt Tree Topology.

Author
Peter Frank Perroni

The communication topology between TH instances must be defined by a THTree instance, where every tree node has its own unique ID, that must match the TH instance ID according to its position in the processing grid.

Therefore, to change TH instance's position in the topology, THTree's node ID must be adjusted accordingly.

Notice it is mandatory to lock() the THTree topology before using it.
