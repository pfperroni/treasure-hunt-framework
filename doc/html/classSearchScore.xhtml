Detailed Description

template<class P = double, int pSize = 1, class F = double, int fSize = 1, class V = double, int vSize = 1>
class SearchScore< P, pSize, F, fSize, V, vSize >

This class stores a Search instance and its score.

Author
Peter Frank Perroni

TH's search group uses the SearchAlgorithmSelectionPolicy to choose and rank the optimization methods.
