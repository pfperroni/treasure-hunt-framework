Detailed Description

template<class F = double, int fSize = 1>
class Fitness< F, fSize >

This structure represents the fitness (or cost) of one Solution.

Author
Peter Frank Perroni

A Fitness on TH can be represented by multiple values, instead of the traditional 1-value fitness (eg. multi-objective optimization, fitness history, score list, etc). The fitness is an ordered list with any number of elements, whose type must be of same basic data type for all elements (eg., double or int).
