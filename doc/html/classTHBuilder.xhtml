Detailed Description

template<class P = double, int pSize = 1, class F = double, int fSize = 1, class V = double, int vSize = 1>
class THBuilder< P, pSize, F, fSize, V, vSize >

Treasure Hunt Framework Builder.

Author
Peter Frank Perroni

Before a TH can be instanced, its configurations must be build by this convenience builder.

Note
For Treasure Hunt Framework, please refer (and cite) to the following:
PERRONI, Peter Frank (2019). Treasure hunt: a framework for cooperative, distributed parallel optimization (Doctoral thesis, UFPR). Institutional repository at UFPR: https://acervodigital.ufpr.br/handle/1884/62517
