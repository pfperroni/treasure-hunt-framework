Virtual method to apply the policy that will relocate TH instance's population at every TH instance's iteration.

A method implementing this virtual method is the responsible for updating the location in the search space of every population individual. It should use the Region (a.k.a. the "anchor") as base to calculate the relocation, as well as the Region::getOriginalDimensions() to limit the overall search space boundaries.

Parameters
relocationStrategyDataThe repository containing useful data to perform the relocation.
regionThe "anchor" Region for current TH instance.
populationThe population, which is comprised of a list of solutions.
