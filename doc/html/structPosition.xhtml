Detailed Description

template<class P = double, int pSize = 1>
class Position< P, pSize >

This structure represents one location within 1-dimension of the search space.

Author
Peter Frank Perroni

A Position on TH (i.e. one location within one actual dimension in the search space) can be represented by multiple values, instead of the traditional 1-value dimension (eg., Fourier-based search space and interval-based search space). The position is an ordered list with any number of elements, whose type must be of same basic data type for all elements (eg., double or int). Therefore, to represent the entire search space, one Position object for every dimension is required.
