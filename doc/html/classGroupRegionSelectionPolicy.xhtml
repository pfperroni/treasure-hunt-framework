Detailed Description

template<class P = double, int pSize = 1, class F = double, int fSize = 1, class V = double, int vSize = 1>
class GroupRegionSelectionPolicy< P, pSize, F, fSize, V, vSize >

This policy implements the sub-region selection criteria by grouping dimensions.

Author
Peter Frank Perroni

All dimensions inside the same group will be partitioned proportionally at same positions, and then an "anchor" sub-region is selected based on TH instance's position on THTree topology.
