template<class P = double, int pSize = 1, class F = double, int fSize = 1, class V = double, int vSize = 1>
class ConvergentBestListUpdatePolicy< P, pSize, F, fSize, V, vSize >

This policy updates the BestList instance by enforcing a behavior that speeds up the convergence, as an attempt to reduce the time required to converge and, consequently, increase communication between TH instances.
